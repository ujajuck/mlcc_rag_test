# 응답 형식 및 전체 응답 예시

스켈레톤 조립 과정의 시각화 형식과 전체 응답 구조 예시를 제공한다.
`search-playbook.md`의 "응답 형식" 섹션과 함께 참고한다.

---

## 스켈레톤 답변 형식 A — 트리 다이어그램

후보 스켈레톤을 아래처럼 트리 다이어그램으로 시각화하면 각 코드 자리의 의미를 직관적으로 전달할 수 있다.

```
CL 03 A 475 M R 3 _ _ _ _
│  │  │ │   │ │ │ │ │ │ └─ 포장 코드 (미확정)
│  │  │ │   │ │ │ │ │ └─── 제어 코드 (미확정)
│  │  │ │   │ │ │ │ └───── 제품/사이즈제어 코드 (미확정)
│  │  │ │   │ │ │ └─────── 설계 코드 (미확정)
│  │  │ │   │ │ └───────── 두께 코드 3 = 0.30 mm
│  │  │ │   │ └─────────── 전압 코드 R = 4 V
│  │  │ │   └───────────── 편차 코드 M = ±20%
│  │  │ └───────────────── 용량 코드 475 = 4.7 μF
│  │  └─────────────────── 온도특성 A = X5R
│  └────────────────────── 사이즈 코드 03 = 0201/0603 (L ≈ 0.60 mm, W ≈ 0.30 mm)
└───────────────────────── "CL" = MLCC 시리즈

* 전체 품번 앞 8자리 고정

완성되지 않은 8~11번째 자리(설계 / 제품·사이즈제어 / 제어 / 포장)는
현재 카탈로그 청크에 근거가 없으므로 _ 로 표시했습니다.
```

## 스켈레톤 답변 형식 B — 번호 주석

후보 스켈레톤의 각 자리를 번호로 분리하고, 아래에 항목별 설명을 나열하는 형식이다.

```
CL   03   A   475   M   R   3   _   _   _   _
──   ──   ─   ───   ─   ─   ─   ─   ─   ─   ─
①    ②   ③    ④    ⑤   ⑥   ⑦   ⑧   ⑨   ⑩   ⑪

① 시리즈     : CL (Standard MLCC)
② 사이즈 코드 : 03 (0.6 x 0.3 mm)
③ 온도특성    : A (X5R)
④ 용량       : 475 (4.7 μF)
⑤ 편차       : M (±20%)
⑥ 정격전압    : R (4 V)
⑦ 두께       : 3 (0.30 mm)
⑧~⑪ 나머지   : 설계 / 제어 / 포장 (카탈로그 데이터 필요)
```

> 형식 A와 B 중 하나를 선택하거나 혼합하여 사용한다. 핵심은 확정된 코드와 미확정 코드를 명확히 구분하고, 각 자리의 의미를 사용자가 한눈에 파악할 수 있게 하는 것이다.

## 전체 응답 예시 — 종합 스펙 사전선정

사용자 프롬프트:

`고객사 의뢰로 스펙 만족하는 MLCC 기종부터 선정해야해. A 온도특성, 정격전압 4V, L size 최대 690um, W size 최대 390um, T size 최대 550um, 기준용량 4.8uF, M편차, 고주파 저전계에서 1V DC 전계를 인가했을 때 최소 3.45uF 를 만족하는 기종을 catalog 기준으로 찾아줘.`

전체 응답:

---

**1. 제약조건 요약**

| 구분 | 항목 | 값 |
|---|---|---|
| hard | 온도특성 코드 | A (X5R) |
| hard | 정격전압 | 4V (code R, 4.0Vdc) |
| hard | L max | 0.690 mm |
| hard | W max | 0.390 mm |
| hard | T max | 0.550 mm |
| hard | 편차 | M (±20%) |
| soft | 기준용량 | 4.8 uF (비표준 → 표준 후보 도출) |
| 검증 전용 | 유효용량 | 고주파 저전계 1V DC에서 최소 3.45 uF |

**2. 스켈레톤 조립 과정**

코드 매핑 (catalog-codebook.md 기반):

- 온도특성 코드: `A` → X5R (Class II, -55~+85°C, ±15%)
- 정격전압 코드: `R` → 4.0Vdc
- 편차 코드: `M` → ±20%
- 사이즈 후보: `03` (0201/0603, 공칭 0.60×0.30 mm) — L/W max 통과
  - `05` (0402/1005, 1.00×0.50 mm) — L/W 초과로 탈락
- 두께 후보: 코드 `3` (0.30 ±0.03 mm) — T max 0.550 mm 통과
- 기준용량: 4.8 uF는 E-series 비표준
  - 후보 1: 5.1 uF → 코드 `515` (상향, 유효용량 마진에 유리)
  - 후보 2: 4.7 uF → 코드 `475` (하향, 타겟에 가장 가까움)

후보 스켈레톤 1 (우선 권장):

```
CL 03 A 515 M R 3 _ _ _ _
│  │  │ │     │ │ │ │   └─ 포장 / 제어 (미확정)
│  │  │ │     │ │ │ └───── 설계 코드 (미확정)
│  │  │ │     │ │ └─────── 두께 코드 3 = 0.30 mm
│  │  │ │     │ └───────── 전압 코드 R = 4 V
│  │  │ │     └──────────── 편차 코드 M = ±20%
│  │  │ └────────────────── 용량 코드 515 = 5.1 μF
│  │  └──────────────────── 온도특성 A = X5R
│  └─────────────────────── 사이즈 코드 03 = 0201/0603 (L ≈ 0.60 mm, W ≈ 0.30 mm)
└────────────────────────── "CL" = MLCC 시리즈
```

→ 5.1 μF는 4.8 μF 타겟 대비 상향 대체로, DC 바이어스에 의한 용량 감소를 고려하면 3.45 μF 최소 요건 충족 가능성이 더 높음.

후보 스켈레톤 2:

```
CL   03   A   475   M   R   3   _   _   _   _
──   ──   ─   ───   ─   ─   ─   ─   ─   ─   ─
①    ②   ③    ④    ⑤   ⑥   ⑦   ⑧   ⑨   ⑩   ⑪

① 시리즈     : CL (Standard MLCC)
② 사이즈 코드 : 03 (0.6 x 0.3 mm)
③ 온도특성    : A (X5R)
④ 용량       : 475 (4.7 μF)
⑤ 편차       : M (±20%)
⑥ 정격전압    : R (4 V)
⑦ 두께       : 3 (0.30 mm)
⑧~⑪ 나머지   : 설계 / 제어 / 포장 (미확정)
```

→ 4.7 μF는 타겟에 가장 가까운 하향 표준 명목이나, 바이어스 감소 후 3.45 μF 미달 리스크 있음.

**3. DB 활성 라인업 히트**

조립된 스켈레톤을 DB 패턴으로 변환하여 조회:

- 패턴: `CL03A515MR3____` → [결과 N건]
- 패턴: `CL03A475MR3____` → [결과 N건]

(결과에 따라 목록 제시 또는 후속 질문)

**4. 검증 필요 항목**

- 1V DC 바이어스 고주파 저전계 조건에서의 실제 유효용량 ≥ 3.45 uF 여부
- 정확한 tail 코드 (설계, 제품/사이즈 제어, 제어, 포장)
- DC 바이어스 특성은 카탈로그의 샘플 그래프만으로는 단정할 수 없으며, 개별 데이터시트 또는 실측 데이터가 필요함
